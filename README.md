# AI Kindergarten Game Project

This project is for creating a playable educational game for kindergarten children.

You will use AI as a programming partner. AI can help explain code, suggest ideas, and build features. You are still the game designer: you decide what children should learn and how the game should feel.

## Our workflow

Idea
↓
Plan
↓
Ask AI for help
↓
Build
↓
Test
↓
Improve

Start with [GAME_PLAN.md](GAME_PLAN.md). Keep your first version small and playable.

## Git basics

Clone your team repository:

```bash
git clone https://github.com/Future-Designer-2026/team-04-game.git
cd team-04-game
```

Save your progress with a commit:

```bash
git add .
git commit -m "Describe what you changed"
```

Send your commit to GitHub:

```bash
git push
```

Make small commits. A small commit is easier to understand, test, and fix than one huge change.
